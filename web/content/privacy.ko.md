# 개인정보 처리방침

*최종 업데이트: 2026년 3월*

## 개인정보 보호에 대한 약속

AISight는 개인정보 보호를 핵심 원칙으로 하여 구축되었습니다. 당사가 운영하는 외부 서버에 귀하의 개인 데이터를 수집, 저장 또는 전송하지 않습니다. 분석, 추적, 광고, 사용자 계정이 없습니다. 귀하의 데이터는 귀하의 기기에 남아 있습니다.

## 수집하지 않는 정보

AISight는 다음을 수집하지 않습니다:

- 개인 식별자(이름, 이메일, 전화번호)
- 추적 목적의 기기 식별자
- 위치 데이터
- 사용 분석 또는 텔레메트리
- 광고 식별자
- 쿠키 또는 브라우저 핑거프린트

당사는 사용자 데이터를 판매, 임대, 공유 또는 수익화하지 않습니다 — 절대로.

## 기기에 저장되는 데이터

AISight는 귀하의 기기에서만 다음 데이터를 로컬로 저장합니다:

**검색 기록:** 귀하의 검색 쿼리와 AI 생성 답변은 Apple의 SwiftData 프레임워크를 사용하여 기기에 저장됩니다. 귀하가 공유하기로 선택하지 않는 한 이 데이터는 기기를 떠나지 않습니다.

**환경설정:** 앱 설정(언어 기본 설정, SearXNG 서버 URL)은 기기의 UserDefaults에 저장됩니다.

설정 → 데이터 → 모든 데이터 삭제에서 언제든지 로컬에 저장된 모든 데이터를 삭제할 수 있습니다.

## 네트워크를 통해 전송되는 데이터

AISight는 두 가지 유형의 네트워크 요청을 합니다:

1. **검색 쿼리:** 검색을 수행하면 귀하의 쿼리가 설정한 SearXNG 서버로 전송됩니다. AISight는 이 서버를 운영하지 않습니다 — 귀하가 제공하고 제어합니다. 검색 쿼리의 개인정보 보호는 SearXNG 인스턴스의 구성에 따라 달라집니다.

2. **웹 콘텐츠 가져오기:** AISight는 검색 결과에서 반환된 URL의 공개적으로 이용 가능한 웹 페이지를 가져와 AI 요약을 위한 텍스트 콘텐츠를 추출합니다. 이는 브라우저에서 웹사이트를 방문하는 것과 유사한 표준 HTTP 요청입니다.

AISight는 iOS/macOS의 URL 로딩 표준을 넘어서는 추적 매개변수, 사용자 정의 헤더 또는 식별 정보를 이러한 요청에 추가하지 않습니다.

## 기기 내 AI 처리

AISight의 모든 AI 처리는 Apple의 FoundationModels 프레임워크(Apple Intelligence)를 사용하여 전적으로 귀하의 기기에서 이루어집니다. 귀하의 쿼리, 검색 결과 및 생성된 답변은 어떤 클라우드 AI 서비스에도 전송되지 않습니다. AI 모델은 기기의 뉴럴 엔진에서 로컬로 실행됩니다.

## 제3자 서비스

AISight 자체는 제3자 분석, 광고, 충돌 보고 또는 데이터 수집 서비스와 통합되지 않습니다.

자체 호스팅하는 SearXNG 인스턴스는 구성 방식에 따라 자체 데이터 관행이 있을 수 있습니다. 자세한 내용은 SearXNG 문서를 참조하시기 바랍니다.

검색 중 가져온 웹 페이지에는 자체 추적 메커니즘이 포함될 수 있지만, AISight는 일반 텍스트 콘텐츠만 추출하며 JavaScript를 실행하거나 추적 리소스를 로드하지 않습니다.

## 아동의 개인정보 보호

AISight는 13세 미만 아동을 포함하여 누구로부터도 의도적으로 개인 정보를 수집하지 않습니다. 개인 데이터를 수집하지 않으므로 관리할 아동 데이터가 없습니다. 그러나 AI가 생성한 콘텐츠가 모든 연령대에 항상 적합하지 않을 수 있습니다. 부모의 지도를 권장합니다.

## 귀하의 권리

AISight는 외부 서버에 귀하의 개인 데이터를 수집하거나 저장하지 않으므로, 많은 전통적인 데이터 권리가 설계상 충족됩니다:

- **접근권:** 모든 데이터는 기기에 로컬로 저장되며 귀하가 접근할 수 있습니다.
- **삭제권:** 설정에서 언제든지 모든 앱 데이터를 삭제하거나 앱을 삭제하여 실행할 수 있습니다.
- **데이터 이동권:** 귀하의 데이터는 기기에 표준 형식으로 저장됩니다.
- **판매 거부권:** 당사는 귀하의 데이터를 절대 판매하지 않습니다.

이러한 권리는 EU 일반 데이터 보호 규정(GDPR) 및 캘리포니아 소비자 개인정보 보호법(CCPA)에 따른 권리를 포함하여 모든 관할권의 사용자에게 적용됩니다.

## 보안

귀하의 데이터는 기기 암호화, 앱 샌드박싱, 생체 인증 또는 암호 잠금을 포함한 Apple 기기에 내장된 보안 기능으로 보호됩니다. AISight는 Apple 프레임워크를 사용하여 기기에서만 데이터를 저장하므로, 귀하의 데이터는 iOS 및 macOS의 전체 보안 아키텍처의 혜택을 받습니다.

SearXNG 인스턴스 및 소스 웹사이트에 대한 네트워크 요청은 서버가 지원하는 경우 표준 HTTPS 암호화를 사용합니다.

## 데이터 보존

검색 기록은 귀하가 명시적으로 삭제할 때까지 기기에 보존됩니다. 당사가 운영하는 외부 서버에는 어떤 데이터도 보존되지 않습니다. 앱을 삭제하면 로컬에 저장된 모든 데이터가 제거됩니다.

## 이 웹사이트

이 웹사이트(private-search-intelligence.app)는 쿠키, 분석 또는 추적을 사용하지 않습니다. 문의 양식은 이메일로 직접 메시지를 전송하며 서버에 데이터를 저장하지 않습니다.
