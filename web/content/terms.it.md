# Condizioni d'Uso

*Ultimo aggiornamento: marzo 2026*

Utilizzando AISight, accetti queste condizioni. Ti preghiamo di leggerle attentamente.

## 1. Accettazione delle Condizioni

Scaricando, installando o utilizzando AISight ("l'Applicazione"), accetti di essere vincolato da queste Condizioni d'Uso. Se non sei d'accordo, non utilizzare l'Applicazione.

## 2. Descrizione del Servizio

AISight è un motore di risposte che cerca nel web tramite un server SearXNG configurato dall'utente e genera risposte sul dispositivo utilizzando Apple Intelligence (framework FoundationModels). L'Applicazione è fornita gratuitamente, senza acquisti in-app né abbonamenti.

## 3. Avvertenza sui Contenuti Generati dall'IA

AISight utilizza l'intelligenza artificiale sul dispositivo per generare risposte basate sui risultati di ricerca web. Riconosci e accetti che:

- Le risposte generate dall'IA possono contenere inesattezze, errori o informazioni obsolete.
- Le risposte non sostituiscono la consulenza professionale di alcun tipo, inclusa ma non limitata a consulenza medica, legale, finanziaria o sulla sicurezza.
- Sei l'unico responsabile della verifica dell'accuratezza di qualsiasi informazione prima di farvi affidamento.
- Le citazioni e le fonti sono fornite come riferimento; dovresti consultare le fonti originali per confermarne l'accuratezza.

L'Applicazione fornisce informazioni "così come sono". Non offriamo garanzie riguardo all'accuratezza, completezza, affidabilità o idoneità dei contenuti generati dall'IA per qualsiasi scopo.

## 4. Configurazione del Server SearXNG

AISight richiede un server SearXNG per funzionare. Sei responsabile di:

- Fornire e mantenere il tuo server SearXNG o utilizzarne uno al quale sei autorizzato ad accedere.
- Assicurarti che il tuo server SearXNG sia conforme alle leggi applicabili e ai termini di servizio dei motori di ricerca che interroga.
- La configurazione di sicurezza e privacy della tua istanza SearXNG.

Non forniamo, gestiamo né garantiamo alcun server SearXNG.

Ti impegni a non utilizzare AISight per:

- Violare leggi o regolamenti applicabili.
