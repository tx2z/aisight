# v1.3 — 새로운 기능

더 똑똑한 답변, 더 적은 오류.

• 이제 답변이 출처를 기반으로 검증됩니다 — 내용이 정확하지 않다고 판단되면 AISight가 자동으로 더 엄격한 근거를 적용하여 답변을 다시 생성합니다
• 새로운 다시 생성 버튼으로 검색 횟수를 소모하지 않고 답변을 다시 받아볼 수 있습니다
• 다시 검색 기능으로 최신 결과를 사용하여 질문을 다시 실행할 수 있습니다
• 출처를 병렬로 불러와 답변이 눈에 띄게 빨라졌습니다
• 개선된 검색 순위로 더 관련성 높은 결과를 보여드립니다
• 향상된 질의 이해로 사용자가 실제로 의도한 내용을 더 정확하게 파악합니다
• VoiceOver가 응답 생성 중임을 안내합니다

모든 새로운 기능은 9개 언어로 제공됩니다.
